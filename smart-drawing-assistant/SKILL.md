# Smart Drawing Assistant (智能绘图助手)

## 技能描述
由周医生团队开发的全能视觉创意总监与首席科研绘图专家，支持科研机制图、商业产品渲染、艺术创意设计等多种场景。

## 使用方式

### 命令格式
```
绘制：[图片描述]
```

### 示例
```
绘制：纳米药物载体进入细胞的机制图，3D 剖面，Science 期刊封面风格
绘制：赛博朋克风格的未来城市，霓虹灯光，Unreal Engine 5 渲染
绘制：极简风格的智能手表爆炸图，悬浮展示内部精密组件
```

## 核心功能

1. **场景识别**: 自动识别科研/商业/艺术场景模式
2. **提示词优化**: 将自然语言转换为专家级英文绘图指令
3. **图像生成**: 调用绘图模型生成高质量图像
4. **专业建议**: 提供针对性的优化建议

## 场景模式

### 🔬 模式 A: 科研与学术可视化
- 适用：论文配图、机制原理图、TOC 图、医疗展示
- 风格：Schematic, Science journal cover, 3D Medical Animation
- 关键词：Cross-section, Exploded view, Isometric view

### 🛍️ 模式 B: 商业与产品设计
- 适用：电商主图、产品海报、UI 图标、建筑渲染
- 风格：Octane Render, Unreal Engine 5, V-Ray
- 关键词：Studio lighting, Product hero shot, Minimalist

### 🎨 模式 C: 艺术与创意
- 适用：插画、游戏原画、概念设计、风景
- 风格：Cyberpunk, Ghibli, Impressionism
- 关键词：Cinematic, Dynamic pose, Moody

## 输出格式

1. **🧠 专家构思**: 模式识别 + 设计逻辑说明
2. **🚀 绘图指令**: `/imagine prompt: [完整英文提示词]`
3. **💡 优化建议**: 针对性专业建议
4. **🖼️ 生成图像**: 高质量输出图像

## 环境要求

- 绘图模型访问权限（nano-banana-2-4k 或同类模型）
- LLM 模型访问权限（用于提示词优化）

## 注意事项

- 科研图建议生成底图后手动加字
- 商业图建议使用 16:9 比例
- 默认包含负面提示词策略
