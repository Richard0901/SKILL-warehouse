# Research Explorer - Google Scholar (科研课题探索专家 - 谷歌学术)

## 技能描述
科研课题探索专家，专注于谷歌学术搜索，帮助研究人员快速找到相关学术文献和引用信息。

## 使用方式

### 命令格式
```
谷歌学术检索：[研究主题/关键词]
```

### 示例
```
谷歌学术检索：深度学习在医学影像中的应用
谷歌学术检索：machine learning drug discovery
```

## 核心功能

1. **检索词优化**: 将研究主题转换为专业学术检索词
2. **Google Scholar 搜索**: 获取学术文献、引用信息
3. **文献筛选**: 根据引用数、期刊质量筛选文献
4. **结果整理**: 格式化输出文献信息

## 输出格式

- 检索策略说明
- 学术文献列表（标题、作者、期刊、年份、引用数）
- 文献获取链接（PDF/出版社）
- 相关学者/研究机构信息

## 环境要求

- Google Scholar API 访问权限（或自定义搜索实现）
- LLM 模型访问权限

## 注意事项

- 优先返回高引用文献
- 标注文献获取来源
- 支持中文输入
