# Tavily Academic Research (Tavily 深度研究 - 学术洞察)

## 技能描述
学术深度研究助手，基于 Tavily 深度搜索结果撰写学术洞察报告，包含 Mermaid 思维导图。

## 使用方式

### 命令格式
```
进行学术研究：[研究主题/课题]
```

### 示例
```
进行学术研究：CRISPR 基因编辑技术的最新进展
进行学术研究：阿尔茨海默病的生物标志物研究
```

## 核心功能

1. **问题优化**: 将用户问题转换为专业英文检索词
2. **Tavily 深度搜索**: 至少检索 50 个网址，获取学术深度信息
3. **知识库检索**: 支持语义检索和全文检索
4. **学术报告生成**: 撰写 5000 字以上深度学术报告
5. **思维导图**: 自动生成 Mermaid 格式思维导图

## 输出格式

- 深度学术研究报告（5000+ 中文字）
- 真实网址链接来源
- PDF 原文链接
- Mermaid 思维导图

## 环境要求

- Tavily API Key
- LLM 模型访问权限（支持 gpt-oss:120b-cloud 或同类模型）
- 知识库（可选）

## 注意事项

- 必须给出真实网址链接来源
- 报告需包含 PDF 原文链接
- 中文字数要求 5000 以上
- Mermaid 思维导图在报告末尾
