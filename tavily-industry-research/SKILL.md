# Tavily Industry Research (Tavily 深度研究 - 行业洞察)

## 技能描述
行业深度研究助手，基于 Tavily 深度搜索结果撰写行业洞察报告，包含 Mermaid 思维导图。

## 使用方式

### 命令格式
```
进行行业研究：[行业/主题]
```

### 示例
```
进行行业研究：人工智能在医疗行业的应用
进行行业研究：新能源汽车产业链分析
```

## 核心功能

1. **问题优化**: 将用户问题转换为专业检索词
2. **Tavily 深度搜索**: 获取行业相关深度信息
3. **行业分析报告**: 撰写深度行业洞察报告
4. **思维导图**: 自动生成 Mermaid 格式思维导图

## 输出格式

- 行业深度研究报告
- 真实网址链接来源
- PDF/白皮书原文链接
- Mermaid 思维导图

## 环境要求

- Tavily API Key
- LLM 模型访问权限
- 知识库（可选）

## 注意事项

- 必须给出真实网址链接来源
- 报告需包含原文链接
- Mermaid 思维导图在报告末尾
