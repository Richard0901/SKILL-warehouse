# Research Explorer - Google (科研课题探索专家 - 谷歌检索)

## 技能描述
科研课题探索专家，使用谷歌搜索引擎进行科研课题相关信息的广泛检索。

## 使用方式

### 命令格式
```
谷歌检索：[研究主题/关键词]
```

### 示例
```
谷歌检索：量子计算最新进展
谷歌检索：gene therapy clinical trials
```

## 核心功能

1. **检索词优化**: 优化用户搜索词，提高搜索精度
2. **谷歌搜索**: 调用 Google Custom Search API
3. **结果筛选**: 筛选科研相关的高质量结果
4. **结果整理**: 格式化输出搜索结果

## 输出格式

- 优化后的检索词
- 搜索结果列表（标题、链接、摘要）
- 相关资源分类整理

## 环境要求

- Google Custom Search API Key
- Custom Search Engine ID (CX)
- LLM 模型访问权限

## 注意事项

- 自动优化检索词以提高搜索质量
- 优先返回学术、政府、机构网站结果
- 支持中文输入
