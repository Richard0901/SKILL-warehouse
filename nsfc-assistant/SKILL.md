# NSFC Assistant (国自然助手)

## 技能描述
国自然课题分析助手，支持知识库检索 +Tavily 深度检索，自动撰写行业洞察报告并绘制 Mermaid 思维导图。

## 使用方式

### 命令格式
```
使用国自然助手分析：[研究课题/关键词]
```

### 示例
```
使用国自然助手分析：肿瘤免疫治疗
使用国自然助手分析：CRISPR 基因编辑技术在临床应用中的进展
```

## 核心功能

1. **问题优化**: 自动将用户问题转换为专业的英文检索词
2. **Tavily 深度搜索**: 至少检索 50 个网址，获取深度搜索结果
3. **知识库检索**: 支持语义检索和全文检索
4. **学术报告生成**: 撰写 5000 字以上深度学术报告
5. **思维导图**: 自动生成 Mermaid 格式思维导图

## 输出格式

- 深度学术研究报告（5000+ 中文字）
- 真实网址链接来源
- PDF 原文链接
- Mermaid 思维导图

## 环境要求

- Tavily API Key
- LLM 模型访问权限（支持 gpt-oss:120b-cloud 或同类模型）
- 知识库（可选）

## 注意事项

- 必须给出真实网址链接来源
- 报告需包含 PDF 原文链接
- 中文字数要求 5000 以上
-  Mermaid 思维导图在报告末尾
