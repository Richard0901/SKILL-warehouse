# 优秀教案撰写指南 (Excellent Lesson Plan Writer)

本技能用于帮助用户撰写结构严谨、逻辑清晰、思政元素融入自然的“优秀教案”。教案框架基于标准模板，可广泛迁移至不同学科、不同案例的教案撰写中。

## 核心框架与撰写要求

教案必须严格遵循以下八大模块结构进行撰写：

### 一、案例背景及主题
- **课程属性与模块**：明确课程的基本属性（必修/选修）、模块分类及授课对象。
- **主题词**：提炼3-6个核心关键词，涵盖专业知识点与核心思政元素。

### 二、结合章节
- **知识内容定位**：明确本案例所属的章、节。
- **章节概述**：简述本章节的核心教学内容。
- **重要性**：阐述该内容在整个课程体系或职业能力培养中的关键意义。

### 三、思政元素
- **元素提炼**：精准提炼与案例高度契合的思政元素（如家国情怀、职业道德、工匠精神、科学精神、集体主义等）。
- **表述方式**：使用高度凝练的短语或短句，将思政内涵与专业精神相结合。

### 四、教学目标
必须包含以下三个维度，且思政目标必须列入素质目标：
1. **（一）知识目标**：学生需要掌握、熟悉、了解的专业理论知识。
2. **（二）能力目标**：学生能够运用知识解决实际问题的能力、实践操作能力或思维能力。
3. **（三）素质目标（课程思政目标）**：结合提炼的思政元素，描述对学生价值观、职业素养、品格情操的塑造目标。

### 五、案例意义
采用“总-分-总”结构阐述：
- **总述**：本次课的核心内容是什么，案例作为载体起到了什么作用。
- **分述**：
  - **第一**：阐述案例如何具象化思政内涵，实现价值塑造与知识传授的同频共振。
  - **第二**：阐述案例如何强化专业认知，推动专业能力与职业精神协同提升。
  - **第三**：阐述案例对学生综合素质（如系统思维、协作精神等）的培养作用。
- **总结**：最终实现“知识传授、能力培养、价值塑造”三位一体的教学目标。

### 六、教学实施
本部分是教案的核心，需详细展开：
1. **（一）课程思政融入的教学内容**
   - **案例背景**：生动、详实地描述案例的背景、经过和结果，突出其中的专业难点与思政闪光点。
   - **知识点与思政的融合**：列出3-5个核心知识点，每个知识点下详细阐述如何将特定的思政元素自然融入（如：知识点A：锚定XX精神）。
2. **（二）教学方法**
   - 列举3-4种教学方法（如案例教学法、多媒体辅助法、互动讨论法、情景模拟法等）。
   - 必须结合本案例具体说明每种方法是如何应用的。
3. **（三）教学理念**
   - 提炼4个核心教学理念（如“战为导向、思政铸魂、知行合一、协同育人”或“学生中心、产出导向”等）。
   - 结合课程特点进行简要阐释。
4. **（四）创新点**
   - 总结3-4个教学创新点（如案例选取的独特性、三维目标的融合度、教学模式的创新、评价体系的完善等）。
5. **（五）教学过程设计（如何契合进教学进程）**
   - 按步骤详细设计教学环节（通常分为4-5步，如：导入案例-理论讲授-案例复盘-总结回顾）。
   - 每一步需明确具体的操作内容，并清晰标注思政元素是如何在这一步中自然融入的。

### 七、教学考核评价
采用表格形式呈现，从三个维度进行评价：
| 考核维度 | 考核方式 | 具体内容 |
|----------|----------|----------|
| 知识掌握 | 课堂提问/测验/作业 | 结合具体知识点说明考核内容 |
| 能力提升 | 实践操作/案例分析/讨论 | 结合具体能力要求说明考核内容 |
| 价值塑造 | 课后反思/课堂表现/报告 | 结合思政目标说明如何评价学生的价值观内化情况 |

### 八、教学效果与反思
1. **（一）教学效果**
   - 从“学员反馈/学生评价”、“能力提升”、“价值认同”三个方面总结教学取得的成效。
2. **（二）教学反思**
   - **优点**：总结本次教学设计的成功之处。
   - **改进方向**：提出2-3条具体、可行的优化建议（如引入更新的案例、增加实践环节时长、丰富评价主体等）。

## 交互工作流 (Workflow)

当用户要求撰写教案时，请遵循以下步骤：

1. **信息收集**：如果用户提供的信息不足，主动询问以下关键信息：
   - 课程名称及授课对象
   - 案例的具体内容或背景
   - 核心专业知识点
   - 希望融入的核心思政元素
2. **大纲确认（可选）**：对于复杂的教案，可以先输出一个简要的八大模块大纲供用户确认。
3. **全文生成**：严格按照上述八大模块的结构和要求，生成完整的教案。
4. **语言风格**：
   - 使用规范、严谨的教学学术语言。
   - 思政融入要自然、“润物细无声”，避免生硬说教。
   - 逻辑清晰，层次分明，多用项目符号和加粗强调重点。
