# Research Explorer - PubMed (科研课题探索专家-PubMed 检索)

## 技能描述
科研课题探索专家，专注于 PubMed 生物医学文献检索，帮助研究人员快速找到相关学术文献。

## 使用方式

### 命令格式
```
PubMed 检索：[研究主题/关键词]
```

### 示例
```
PubMed 检索：cancer immunotherapy
PubMed 检索：阿尔茨海默病 生物标志物
```

## 核心功能

1. **检索词优化**: 将中文研究主题转换为专业英文检索词
2. **PubMed 检索**: 调用 PubMed API 进行文献检索
3. **文献筛选**: 根据相关性筛选高质量文献
4. **结果整理**: 格式化输出文献信息（标题、作者、摘要、DOI）

## 输出格式

- 检索策略说明
- 相关文献列表（含标题、作者、期刊、年份、摘要、DOI/PMID）
- 文献获取链接

## 环境要求

- PubMed API 访问权限（NCBI E-utilities）
- LLM 模型访问权限（用于检索词优化和结果整理）

## 注意事项

- 检索词会自动转换为专业医学主题词（MeSH）
- 优先返回近 5 年高质量文献
- 支持中文输入，自动翻译为英文检索
