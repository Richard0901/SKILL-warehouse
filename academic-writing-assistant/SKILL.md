# Academic Writing Assistant (学术论文写作助手 - 谷歌学术)

## 技能描述
学术论文写作助手，基于谷歌学术搜索结果辅助用户撰写学术论文。

## 使用方式

### 命令格式
```
论文写作辅助：[论文主题/研究方向]
```

### 示例
```
论文写作辅助：人工智能在医疗诊断中的应用
论文写作辅助：review of CRISPR technology
```

## 核心功能

1. **文献检索**: 通过谷歌学术搜索相关文献
2. **文献综述**: 基于搜索结果撰写文献综述
3. **写作建议**: 提供论文结构和写作建议
4. **引用整理**: 整理相关文献的引用信息

## 输出格式

- 相关文献列表
- 文献综述草稿
- 论文结构建议
- 引用格式参考

## 环境要求

- Google Scholar API 访问权限
- LLM 模型访问权限
- 知识库（可选）

## 注意事项

- 提供的引用需用户核实
- 建议仅供参考，需结合具体研究内容
- 支持中英文写作辅助
