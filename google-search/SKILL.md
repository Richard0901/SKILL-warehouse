# Google Search (谷歌搜索)

## 技能描述
谷歌搜索引擎集成，支持自定义搜索词，自动获取搜索结果并格式化输出。

## 使用方式

### 命令格式
```
搜索：[搜索关键词]
```

### 示例
```
搜索：Python 机器学习教程
搜索：climate change research 2025
```

## 核心功能

1. **问题优化**: 优化用户搜索词
2. **Google Custom Search API**: 调用谷歌自定义搜索接口
3. **结果格式化**: 格式化输出搜索结果（标题、链接、摘要）

## 工作流程

1. 接收用户搜索词
2. 优化搜索词（可选）
3. 调用 Google Custom Search API
4. 格式化返回结果

## 输出格式

```
【搜索结果】

[1] 标题
链接：URL
摘要：内容摘要

[2] ...
```

## 环境要求

- Google Custom Search API Key
- Custom Search Engine ID (CX)
- LLM 模型访问权限（可选，用于问题优化）

## 注意事项

- 需要配置 API Key 和 CX
- 默认返回前 10 条结果
- 可配置搜索深度
